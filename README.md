-Water Tracker

A simple and modern daily water tracking application. The user sets a daily goal, adds +200 ml water with each button click, and the progress bar fills up with a smooth animation. At the end of the day, the consumption is automatically saved to history.

-Features 

Set daily water intake goal

+200 ml quick add button

Animated progress bar

Daily history tracking (LocalStorage)

Motivational messages

Modern, responsive UI design


-Technologies Used

HTML

CSS

JavaScript

LocalStorage API
